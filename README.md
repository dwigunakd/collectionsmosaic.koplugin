# Collections View - Koreader Plugin 
## A KOReader plugin that transforms collections into a more visual, folder-like browsing experience. Tested on Kindle Paperwhite 12 Koreader 03.2026
-----
## Features 
-----
- ### **Browse collections as virtual folders**
	Each collection appears as its own folder, with a background image automatically generated from the book covers inside.
- ### **Custom sorting within collections**
	Arrange books inside each collection using your own preferred order.
- ### **Rename collections**
	Easily rename collections directly from the interface.

## Installation
-----
1. Download the Zip Below
2. Extract and copy the zcollectionsview.koplugin folder to your device's plugins/ directory
3. Restart KOReader and go to **Menu -> Tools -> Collection** View to change the settings
